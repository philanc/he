# he

A small bunch of general purpose Lua utility functions

* string extensions (split, strip, ...)
* table and list (map, filter, extend, ...)
* OS and file (ISO date and time, read and write file, shell, ...)
* parsing and encoding (CSV, ...)
* etc.
